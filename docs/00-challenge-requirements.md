# Challenge requirements snapshot

This file summarizes the challenge text supplied on 2026-07-17. Re-check the
official challenge page before submission because terms and deadlines can change.

## Challenge

Build a custom MCP server that connects Kiro to a real external tool, API, data
source, local system, device, or service, and then build something whose value
depends on that integration.

Unexpected and fun integrations are encouraged.

## Required submission materials

- working custom MCP server;
- meaningful use of the MCP integration;
- real system connection;
- public GitHub repository;
- MCP server code;
- `.kiro` folder;
- README explaining connection, setup, and project;
- no committed secrets;
- demo video aiming for 30–60 seconds and no longer than 3 minutes;
- video showing the complete flow from MCP server connection to final output.

## Does not count

- only using a prebuilt community MCP server;
- a project that does not meaningfully use MCP;
- hardcoded or fake data instead of a real system.

9:59 AM GMT+3.
